# yocto_env_app_dev
このプロジェクトは、Yocto環境の構築とラズベリーパイ４用のアプリ開発について記載しています
## Yocto_raspi_dev.md
本ファイルは、Yocto環境の構築方法と開発したラズベリーパイ４用のアプリについて記載しています
## png_file
本デイレクトリは、Yocto_raspi_dev.md 内で使用する画像ファイルを格納しています
## build/conf
本ディレクトリは、ビルド環境全体の設定を管理する場所です
## meta-custom
本ディレクトリは、ユーザーが独自に作成・追加するカスタムレイヤーでレシピやアプリやドライバを格納しています
